# IIC3103-T02 - Rafael Abusleme

## Build Setup

### Prerequisites

- Node.js Version >= v12 - (https://nodejs.org/)
- Yarn - (https://classic.yarnpkg.com/en/docs/install)
- Postgres

### How to run the app

```bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:5000
$ yarn dev
```
